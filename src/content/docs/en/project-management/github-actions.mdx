## Terraform deployment
All Terraform projects in the data work space have a terraform run for a PR. This should be based on Directory and Project relationship. There is a lot of unnecessary runs on each commit. Typically one one directory is modified but 5+ Terraform project runs.

## Tableau Deployment
PR commits trigger modifications of Tableau. User Groups, tbw deployment, and permissions are managed through the repo.

## Wheel version management

Anytime the library directory is modified the python wheel is created and saved to the databricks dbfs.
