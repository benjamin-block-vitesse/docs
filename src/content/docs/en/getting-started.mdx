<h2>What is Data Hub?</h2>

The Data Hub is an **all-in-one** location for learning about existing data assets and planned activity.

## Key Areas

- **[data model documentation](https://):** Definitions and linage of metrics used in analytics and visualization.
- **[Data quality test records](https://kind-forest-03dc9c310.3.azurestaticapps.net/gx):** The testing suite prevents a single data error from repeatedly impacting the business.
- **Product Features:** Core features are often delivered through the data architecture using the lakehouse and visualization tools.
- **Business Domains:** Vitesse applications, salesforce, duco, and other data sources are combined and cleaned to from domains based on the business case.
